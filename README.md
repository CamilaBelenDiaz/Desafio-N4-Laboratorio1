# Desafio-N4-Laboratorio1
Ejercicio práctico de Desarrollo Web: creación de carpeta y archivo index.html con título “Laboratorio 1”. Se aplican etiquetas semánticas (listas, strong, párrafos, imágenes) y tipografía Montserrat desde Google Fonts en variantes 700 y 400. Incluye h1 y h2 con tamaños específicos, color tomado con Colorcop y correo electrónico anidado en h2.
